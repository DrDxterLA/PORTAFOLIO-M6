# Portafolio Personal - Vue 3

## Descripción

Este proyecto corresponde a la actualización de un portafolio web previamente desarrollado en HTML, CSS y JavaScript.

En esta nueva versión se implementa Vue.js para aprovechar sus capacidades reactivas, organización mediante componentes y navegación dinámica utilizando Vue Router.

## Tecnologías utilizadas

* Vue 3
* Vue Router
* Bootstrap 5
* Bootstrap Icons
* JavaScript ES6
* Vite

## Características del proyecto

* Aplicación SPA (Single Page Application)
* Navegación dinámica con Vue Router
* Componentes reutilizables
* Datos reactivos usando v-for
* Diseño responsive con Bootstrap

## Instalación del proyecto

Clonar repositorio

git clone repositorio

Entrar al proyecto

cd portafolio-vue

Instalar dependencias

npm install

Ejecutar servidor de desarrollo

npm run dev

Abrir en navegador

http://localhost:5173
